Before the run starts, you define what the agent is allowed to do. If it goes off-task, TraceMemory catches it before any damage is done.

TraceMemory monitors tool call sequences against the task contract in real-time. If the agent starts crawling internal docs it shouldn't, we pause the run.

Core concepts

Run
A single instantiation of an agent task containing one or more checkpoints.
Task contract
The approved tool scope and goal definition enforced at runtime.
Event
Any interaction within a session, including model calls and tool executions.
Tool trace
Evidence of a tool's invocation, input parameters, and raw return value.
Checkpoint
A durable, cryptographically signed state persisted to PostgreSQL.
Context Health
A metric comparing the current run drift against the task contract.
Receipt
An Ed25519-signed export of a completed run, verifiable offline.
