# Mock MCP Tools

These keyless tools simulate the external evidence used by the hackathon demo:

- `fetch_support_tickets` returns paginated support-ticket evidence.
- `fetch_policy_context` returns fresh and stale policy sources for Context Health.
- `fetch_compliance_tickets` returns blockers after the task is modified.

They keep the demo open-source, deterministic, and runnable without third-party API keys.
